#### Quick start:

1. **Clone the repository:**
```bash
git clone https://github.com/MrCrowNFT/hiredge-ai
cd hiredge-ai

```
2. **Create a virtual environment**
```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

```

**Run the app**

```sh
python3 manage.py runserver
```
